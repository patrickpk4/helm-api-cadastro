Após instalar o chart Helm da aplicação cadastro de produto, siga as instruções abaixo para acessar o sistema.

 Acesso à aplicação

A aplicação ficará disponível em:

http://cadastro.local:30080/swagger


Esse endereço assume que:

Você configurou o DNS local (ex.: /etc/hosts, CoreDNS ou outro método) apontando cadastro.local para o IP do seu node ou LoadBalancer.

A aplicação está exposta na porta 30080 via NodePort ou equivalente.

Exemplo para adicionar no /etc/hosts:

127.0.0.1   cadastro.local


ou, no caso de cluster remoto, substitua pelo IP do node:

192.168.x.x   cadastro.local

 Recursos publicados

Swagger UI: /swagger

API base: /api

 Verificação rápida

Use o comando para verificar o serviço:

kubectl get svc -n <seu-namespace>


Verifique se o tipo é NodePort, LoadBalancer ou se utiliza Gateway/HTTPRoute de acordo com sua configuração.
